# 03 - Demand Forecasting

**Business Problem:** "Gelecek ay ne kadar satacağız?"

## Teknik İçerik
- **Zaman Serisi Analizi:** Trend, Mevsimsellik, Döngüsellik analizi.
- **Modeller:** ARIMA, SARIMA, Prophet, LSTM (Deep Learning opsiyonel).

## Hedeflenen Çıktılar
- Gelecek dönem talep tahmini.
- Stok optimizasyonu için girdi (Safety Stock belirleme).
- Out-of-Stock (Yok satma) durumunu engelleme.

## Kullanılacak Kütüphaneler
- statsmodels, prophet, tensorflow/keras (opsiyonel)
