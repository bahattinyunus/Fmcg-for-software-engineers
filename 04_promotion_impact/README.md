# 04 - Promotion Impact Analysis

**Business Problem:** "Bu kampanya gerçekten işe yaradı mı?"

## Teknik İçerik
- **Uplift Modeling:** Promosyonun satış üzerindeki saf etkisini ölçme.
- **A/B Testi Mantığı:** Kampanyalı dönem vs Kampanyasız dönem (veya benzer ürünler arası karşılaştırma).
- **Cannibalization Analizi:** Bir ürünün promosyonu diğer ürünün satışını yedi mi?

## Hedeflenen Çıktılar
- Kampanya ROI (Yatırım/Getiri) hesabı.
- Gelecek kampanyalar için stratejik öneriler.

## Kullanılacak Kütüphaneler
- scipy (istatistik testleri), pandas
