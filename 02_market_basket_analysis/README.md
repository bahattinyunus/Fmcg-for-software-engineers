# 02 - Market Basket Analysis

**Business Problem:** "Müşteri sepetinde ürünler nasıl eşleşiyor?"

## Teknik İçerik
- **Association Rule Learning:** Birliktelik kuralları öğrenimi.
- **Apriori Algoritması:** Sık geçen ürün kümelerinin bulunması.
- **Metrikler:** Support (Destek), Confidence (Güven), Lift.

## Hedeflenen Çıktılar
- **Cross-sell:** X alan Y de alır (Örn: Cips alan Cola da alır).
- **Bundle Önerileri:** Promosyon paketleri oluşturma.

## Kullanılacak Kütüphaneler
- mlxtend, pandas
