# 05 - KPI Dashboard

**Business Problem:** "Yönetici sabah ekranda ne görmek ister?"

## Teknik İçerik
- **Veri Görselleştirme:** Etkili ve sade grafikler.
- **KPI Tanımları:** Net Satış, Brüt Kar, Sell-Through Rate, Stok Devir Hızı.
- **İnteraktiflik:** Filtreleme ve drill-down özellikleri.

## Hedeflenen Çıktılar
- Yönetim karar destek ekranları.
- Anlık performans takibi.

## Kullanılacak Kütüphaneler
- Streamlit, Dash veya PowerBI/Tableau (konsept olarak)
