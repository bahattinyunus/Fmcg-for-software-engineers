# 01 - Sales Analysis

**Business Problem:** "Hangi ürün, nerede, ne zaman satıyor?"

## Teknik İçerik
- **EDA (Exploratory Data Analysis):** Veri setini anlama, eksik veri analizi, aykırı değer tespiti.
- **Trend Analizi:** Satışların zaman içindeki değişimi (Haftalık, Aylık, Mevsimsel).
- **Kategori/Bölge Kırılımı:** En çok satan kategoriler ve bölgelerin tespiti.

## Hedeflenen Çıktılar
- Düşük performanslı ürünlerin belirlenmesi (Delist önerileri).
- Satışın yoğunlaştığı zaman dilimlerinin analizi.

## Kullanılacak Kütüphaneler
- Pandas, Matplotlib, Seaborn
