## How to Contribute

1.  **Fork the repository** to your GitHub account.
2.  **Clone the fork** to your local machine.
3.  **Create a branch** for your fix or feature (`git checkout -b feature/amazing-feature`).
4.  **Make your changes** and commit them (`git commit -m 'Add some amazing feature'`).
5.  **Push to the branch** (`git push origin feature/amazing-feature`).
6.  **Open a Pull Request** to the `main` branch.

## Code Style

- Use Python PEP 8 standards.
- Write clear comments and documentation.

## Reporting Bugs

Please inspect the issue tracker to see if the bug has already been reported. If not, please open a new issue.
